Что такое SELECT в SQL?
A. Команда для вставки данных
B. Команда для выборки данных
C. Команда для удаления данных
ANSWER: B

Для чего используется WHERE?
A. Для сортировки результатов
B. Для фильтрации строк по условию
C. Для группировки строк
ANSWER: B

Что означает NULL в SQL?
A. Пустая строка
B. Нулевое число
C. Отсутствие значения
ANSWER: C

Какой оператор используется для поиска по шаблону строки?

C. LIKE
ANSWER: C

C. Фильтрует дубликаты

B. Для группировки строк с одинаковыми значениями
C. Для объединения таблиц

Чем HAVING отличается от WHERE?
A. HAVING фильтрует группы, WHERE фильтрует строки
B. HAVING работает быстрее WHERE
C. Это полные синонимы
